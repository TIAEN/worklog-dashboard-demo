# Worklog Dashboard Demo

一个静态工作日志仪表盘示例，包含分类导航、搜索、时间热力图、热力词图、时间线和详情面板。

本仓库只包含前端网站和虚构示例数据，不包含真实工作记录、个人路径、邮箱、姓名或私有项目内容。

## 本地预览

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:8765/`。
